## User Management List | Sample Project
Project developed with node v16.15.1 & npm 8.11.0

## How to 
```bash
# run on localhost:3003
npm run dev

# export as SPA
npm run export
# copy files under /out folder to your hosting

# Run in production mode - SSR
npm run build
npm run start
```
